How to Use TruthBuy extension 
Extract the zip file Truthbuy
Install the Extension: The extension needs to be loaded into your browser (typically via "Load unpacked" in Chrome's extensions management page).
Navigate to an Amazon Product Page: Go to any product details page on a supported Amazon domain (e.g., amazon.com, amazon.in, amazon.co.uk, etc.).
Click the Extension Icon: Click the TruthBuy icon (a white 'T' on a blue background) in your browser's toolbar.
